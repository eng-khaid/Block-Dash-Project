# Block Dash Project

## Project Title

Block Dash Pro

## Project Overview

Block Dash Pro is a lightweight HTML5/JavaScript browser game where players dash across platforms, avoid hazards, and aim for high scores. Built for quick deployment and easy sharing, the project demonstrates core game mechanics, responsive controls, and clean UI design suitable for a portfolio showcase.

## Features & Gameplay Mechanics

- Fast-paced platform-dash gameplay
- Dash / jump mechanic with responsive timing
- Increasing difficulty and obstacle patterns
- Score tracking and session-based leaderboards (local)
- Simple, minimal visuals for clarity and performance

## Technologies Used

- HTML5 (Canvas / DOM)
- CSS3
- JavaScript (ES6+)
- Optional: images and audio assets included in `assets/` (if present)

## Installation & Usage

1. Clone the repository or download the project folder.
2. Open `Block Dash Pro.html` in a modern browser to run the game, or serve the folder with a local HTTP server to avoid file protocol restrictions.

Quick local server options:

```bash
# Python 3
python -m http.server 8000
# Then open: http://localhost:8000/Block%20Dash%20Project/Block%20Dash%20Pro.html
```

```bash
# Node (serve)
npm install -g serve
serve "Block Dash Project"
```

## Game Controls

- Move left: Left Arrow or `A`
- Move right: Right Arrow or `D`
- Jump / Dash: Spacebar or `W` / Up Arrow
- Pause: `P` (if implemented)

See the game scripts (`script.js` or similarly named files) for exact control mapping.
