## Project Overview

Block Dash Pro is a lightweight browser-based platform game built using HTML5 Canvas and JavaScript. The player must jump over obstacles, survive increasing difficulty, and reach the finish line while achieving the highest score.

This project demonstrates core game development concepts such as game loops, collision detection, physics simulation, and interactive UI design.

---

## Features

Fast-paced platform jumping gameplay
Smooth movement and physics-based controls
Progressive difficulty increase
Score tracking system
Victory screen with restart option
Lightweight and runs directly in the browser

---

## Technologies Used

HTML5 (Canvas)
CSS3
JavaScript (ES6+)

## Controls

Space / W / Up Arrow → Jump
A / D / Left / Right Arrows → Move (if enabled)

## How to Run

1. Download or clone the repository
2. Open Block Dash Pro.html in a browser
3. Or use a local server:
   python -m http.server 8000
4. Open http://localhost:8000 in your browser
